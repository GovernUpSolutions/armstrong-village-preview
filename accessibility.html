Accessibility

Last updated June 28, 2026

Armstrong Village Homeowners Association wants residents and visitors to be able to use this website in a practical and accessible way.

Accessibility approach

This site is designed as a simple static website with readable text, keyboard-friendly links, descriptive page titles, alt text for meaningful images, and layouts that adapt to smaller screens. The site aims to support commonly accepted accessibility practices, including WCAG 2.1 AA principles where practical for a small community website.

Known limitations

Some documents, downloaded files, embedded images, or third-party services may not meet the same accessibility standard as the main website. This includes content hosted by TownSq, Facebook, email providers, or document files that were created outside this website.

Requesting help or an alternate format

If you cannot access information on this website, the board can help route the request or provide information in another reasonable format when available. Please include the page or document name, the issue you encountered, and the format that would work better for you.

Contact

For accessibility feedback or assistance, contact the board at armstrongvillagehoa@gmail.com.
